# DBuzz Sign Up Client App and Backend
### Developed by **[aaashir](https://github.com/aaashir)**

DBuzz Sign Up Client App and Backend for Join.D.Buzz

## Installation ⚙
```bash
> git clone https://github.com/d-buzz/d.buzz-signup-app.git
> yarn install
> yarn start
```

## Build ⚙
```bash
> yarn build
```
